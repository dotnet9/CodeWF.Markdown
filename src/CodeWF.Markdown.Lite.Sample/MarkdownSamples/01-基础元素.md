# 01 基础元素

这份文件用于检查 `CodeWF.Markdown.Lite` 的常用 Markdown 渲染。Lite 版本只保留基础阅读能力，不包含代码高亮、数学公式、SVG 预览、脚注回跳或多语言资源。

## 标题与段落

标题、段落和行内样式应保持清晰的层级。正文可以包含 **加粗文本**、*斜体文本*、`inline code` 和普通链接样式：[Avalonia](https://avaloniaui.net)。

长段落用于观察中文换行效果：这一段包含连续的中文说明、英文包名 `CodeWF.Markdown.Lite`、数字版本号和标点符号，预览区应自然换行，不应出现横向撑破窗口或文字重叠。
